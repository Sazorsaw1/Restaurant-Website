Access Summary

Admin

Orders, staff-pick voting, menu management, and user management.

Chef

Orders, staff-pick voting, and menu updates including add, archive, and restore.

Staff

Orders plus staff-pick voting, with no access to menu or user controls.

Order Management

Track the active queue, move orders through service statuses, and review completed orders in a separate history view.

Chef's Pick Voting

Each staff account can keep up to three active votes. The three menu items with the most votes will be marked as Chef's Pick on the customer homepage.

Menu Management

Chef and admin can add menu items with an image path or URL, then update category and price directly from the catalog list. Removing uses archive/restore so items can be safely brought back later.

User Management

Only admins can create staff and chef accounts for the staff-only dashboard.
